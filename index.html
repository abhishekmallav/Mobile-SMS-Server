Local Infrastructure

Turn your smartphone into a local SMS gateway.

Connect to your Android phone via REST API to programmatically send SMS messages over WiFi. A free, 100% private alternative to cloud APIs, powered by Python, Kivy, and Flask.

The Intuition

Why pay for cloud SMS APIs when you already have a phone plan?

Zero Cloud Costs

During feature development, developers often need to test automated SMS messages. Utilize your unused mobile recharge SMS quotas instead of getting charged per message by cloud platforms.

100% Local & Private

The app operates entirely on your local network. No external servers involved, keeping all your data and phone numbers off third-party databases.

Developer Ready

With a built-in Flask background service, it exposes standard HTTP POST REST endpoints directly from your phone to your local dev environment.

Core Features

Engineered for local networking

RESTful API

Simple `POST /api/sms` bindings expecting JSON payloads. Easy to integrate via curl, requests, or Postman.

Interactive UI

A clean Kivy dashboard directly on your Android device displaying the local WiFi IP address and a real-time HTTP interaction log.

Native Integration

Bridges Python natively to Android Java APIs (`SmsManager`) using PyJNIus. Includes persistent background notifications to keep the server alive.

Mobile SMS Server

A lightweight, Python-powered local gateway to bridge native Android capabilities into your daily development workflow.
